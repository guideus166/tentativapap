    • Sobre o projeto

    O projeto surgiu a partir de uma sugestão do professor Bruno Leite, meu orientador, para explorar a impressora 3D da escola — uma Creality K1. Trata-se de um modelo acessível, com várias vantagens como estrutura fechada, bico de alta temperatura, extrusora direct drive dual gear e velocidade máxima de 600 mm/s. A partir disso, tive a ideia de criar um manual que permita a qualquer professor, mesmo sem formação técnica, utilizar a impressora com facilidade. Isso amplia as possibilidades de ensino, oferecendo novas formas de aprendizagem aos alunos. Para complementar, desenvolvemos também um site para apresentar o projeto e, mais adiante, um robô impresso em 3D.
